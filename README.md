## Описание проекта

Выполнен комплексный анализ небольшого корпуса текстов на английском языке (15 рассказов Эдгара Аллана По).

**Выполненные шаги:**

1.  **Извлечение текста** из PDF (`pdftools`) + демонстрация предобработки изображений (`magick`) для улучшения качества (deskew, контраст, нормализация).

2.  **Очистка текста** регулярными выражениями (`stringr`): исправление артефактов, нормализация, удаление шума.

3.  **Лемматизация и частотный словарь** с помощью модели `udpipe` (english-ewt-ud-2.5-191206.udpipe). Учитывались NOUN, ADJ, VERB.

4.  **Коллокации** (биграммы) через `quanteda`, визуализация:

-   Столбчатая диаграмма топ-20
-   Сетевой график топ-30 (`ggraph`)
-   Таблицы результатов

**Ключевые результаты:**

-   Доминирующие темы: смерть, дом, время, глаз, сердце, страх — классические мотивы готической прозы По.

-   Значимые коллокации: "red death", "house usher", "black cat", "tell tale".
