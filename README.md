# Комплексный анализ корпуса рассказов Эдгара Аллана По (Вариант 1)

Дей Захар 

## Описание проекта
Корпус состоит из 15 рассказов Эдгара Аллана По на английском языке в формате PDF.  
Выполнен OCR с помощью tesseract, текст очищен с помощью регулярных выражений, лемматизация и частотный анализ с помощью udpipe, поиск и визуализация коллокаций.
